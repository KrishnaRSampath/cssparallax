Apprentice100

I'm the Program Director of the A100 Program -- a software developer talent accelerator. In a three month cycle, we gather together a Cohort of 20-25 CS majors to train them in the craft of software development.

For the first month, we work collaboratively on projects like this site, using HTML, CSS, JavaScript, and Twitter Bootstrap.

In the second month, we introduce them to the back end side of web application development, focusing on integrating PHP and MySQL with a minimal, effective front end.
